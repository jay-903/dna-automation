## üöÄ Getting Started
To get started, follow these steps in detail:

1. **Install a Compatible Browser**: Ensure you have a modern web browser such as Google Chrome or Firefox. These browsers support the needed features for running the scripts effectively.

2. **Open the Game**: Navigate to the Duet Night Abyss game in your browser, and log in to your account.

3. **Access the Console**: 
   - For Chrome: Right-click anywhere on the page, select "Inspect," and click on the "Console" tab.
   - For Firefox: Right-click, select "Inspect Element," and then click on the "Console" tab.

4. **Copy the Script**: After you download the script file, open it to view its contents. Click inside the console and paste the script directly from your text editor.

5. **Run the Script**: Hit "Enter" in the console to execute the script. You should see the effects immediately in the game.
